# Achievement #1 — «Параллельный таймер» (AVR Assembler)

## Платформа
- МК: **ATmega8**
- Частота: **16 МГц**
- Среда: **Atmel Studio 7.0**
- Ассемблер: **avrasm2**
- Device Pack: **ATmega_DFP**

---

## Задание
Разработать программу на **AVR Assembler**, в которой:
- работают **два параллельных таймера**;
- интервалы задаются константами `TIMER1_INTERVAL` и `TIMER2_INTERVAL`;
- при срабатывании таймеров в **USART** выводятся строки:
  - Timer0 → `ping\r\n`
  - Timer2 → `pong\r\n`.

---

## Реализация
- Код находится [здесь](https://github.com/BashkaCoder/DCS_Achievements/blob/main/Achievement_1/main.asm)
- Использованы таймеры:
  - **Timer0** — прерывание переполнения (`0x0009`);
  - **Timer2** — прерывание переполнения (`0x0004`).
- Оба таймера:
  - режим **Normal**;
  - делитель частоты **1024**.
- Вывод в USART выполнен в **обработчиках прерываний**.
- USART работает в **опросном режиме** (без прерываний).

---

## Средняя сложность
Требовалось найти минимальные интервалы таймеров, при которых программа работает корректно.

### Расчет интервалов

Таймеры настроены с делителем частоты **1024**.  
При частоте микроконтроллера **16 МГц** это означает, что таймер увеличивает счётчик
**15 625 раз в секунду**.

Так как таймеры 8-битные, переполнение происходит после **256 тиков**.  
Следовательно, одно переполнение таймера происходит примерно раз в:

```
256 / 15625 ≈ 16.384 мс
```

---

При срабатывании таймера в обработчике прерывания выводится строка в USART:

- строка `"ping\r\n"` или `"pong\r\n"` содержит **6 символов**;
- USART работает на **9600 бод**, формат **8N1**;
- передача одного символа занимает примерно **1.04 мс**;
- полный вывод строки занимает около **6.25 мс**.

Даже если оба таймера сработают практически одновременно:
- их обработчики выполняются **последовательно**;
- суммарное время вывода двух строк составляет около **12.5 мс**;
- это значение **меньше периода переполнения таймера (16.384 мс)**.

Следовательно, процессор успевает полностью обработать оба прерывания
до следующего переполнения таймера, и программа работает корректно.

На основании этого минимально возможные интервалы выбраны равными **1**.

### Итог
Реализована параллельная работа двух таймеров ATmega8 с выводом данных через USART.  
Задание стандартной и средней сложности выполнено.
